SHORT DESCRIPTION
    KaguwaTools is a set of tools for day-to-day administrative tasks.

LONG DESCRIPTION
    KaguwaTools is a set of tools for day-to-day administrative tasks. 
    Primary the Cmdlet Get-NetworkConnection.

    You can find full documentation here:
        https://github.com/KarlGW/KaguwaTools

DETAILED DESCRIPTION
    KaguwTools is a module containing a set of Cmdlets for day-to-day administrative tasks.
    
    Cmdlets

    Get-NetworkConnection - Returns a list with Kaguwa.Tools.Network.NetworkConnection objects.
    These represent TCP/UDP connections on the local host.
